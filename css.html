Learn CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance of HTML elements. It’s essential for making webpages visually appealing and responsive.

Why Learn CSS?

  • Style your webpages with colors, fonts, and layouts.
  • Make your website responsive and mobile-friendly.
  • Bring your designs to life with animations.

Getting Started

You can add CSS to your webpage in three main ways:

  1. Inline: Directly within an HTML tag using the style attribute.
  2. Internal: Inside a <style> tag in the HTML <head>.
  3. External: In a separate file linked with the <link> tag.

Example: Styling Text

Here’s an example of how CSS can style a webpage:

            
            <style>
                h1 {
                    color: #3498db;
                    font-size: 2.5rem;
                }
            </style>
